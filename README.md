# Survey Akademik Gen Z

Aplikasi web survey ini dikembangkan untuk mengumpulkan data penelitian mengenai **peran pola asuh orang tua dalam memprediksi motivasi belajar mahasiswa Generasi Z semester awal**.

Survey ini dirancang berbasis web, ringan, responsif, dan terintegrasi langsung dengan **Google Spreadsheet** melalui **Google Apps Script** sebagai backend penyimpanan data.
